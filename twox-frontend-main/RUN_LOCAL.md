# Run frontend locally (Windows)

## Prereqs
- **Node.js 20.x** installed

## 1) Configure environment
- `.env.local` is created for you.
- Ensure `NEXT_PUBLIC_BACKEND_API` points to your backend, e.g. `http://localhost:5000/api`.

## 2) Install and run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
